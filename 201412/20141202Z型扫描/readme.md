//规律
从1开始，
每当在第一行且不是最后一列时：行数不变，列数加1 ；然后向下折返；
	向下折返：行数加1，列数减1；
当在第一列且行数不是第一行也不是最后一行时：行数加1，列数不变；然后向上折返；
	向上折返：行数减1，列数加1；
若在第一行且是最后一列时，行数加1，列数不变；然后向下折返；
当在第一列且是最后一行时，行数不变，列数加1；然后向上折返；
当在最后一行且在最后一列时，结束。
